隱私權政策 — NotebookLM Omni-Bridge

最後更新：2026-04-21

NotebookLM Omni-Bridge（下稱「本擴充功能」）是一個開源 Chrome 擴充功能， 用途是將使用者目前瀏覽的網頁內容送進 Google NotebookLM，並在 Chrome 側邊欄提供對話介面。

一、我們不收集任何個人資料

  • 本擴充功能沒有開發者自有的伺服器，也不會將任何資料傳送到第三方。
  • 我們不會收集、儲存或傳輸：姓名、Email、IP 位址、瀏覽紀錄、裝置識別碼、或任何可識別個人身分的資訊。
  • 我們不使用 Analytics、埋點、廣告識別碼或任何追蹤工具。

二、資料流向

本擴充功能的所有網路請求，皆直接發送至 Google NotebookLM （https://notebooklm.google.com），並使用您 Chrome 瀏覽器中既有的 Google 登入 Cookie 進行驗證。此行為等同於您自己在瀏覽器中開啟 NotebookLM 網站。

以下內容會由您的瀏覽器傳送至 Google：

  • 您主動點擊「吸取」時，當前頁面的標題、網址與文字內容（或您所選取的文字）
  • 您在對話框中輸入的訊息
  • 您選擇要上傳的檔案內容

這些資料由 Google 依其自身的 隱私權政策處理。本擴充功能開發者無法取得這些資料。

三、本機儲存

本擴充功能使用 chrome.storage 將下列資料僅儲存於您的本機瀏覽器：

  • 使用者偏好設定（介面主題、預設 Notebook、UI 語系等）
  • 對話歷史記錄（僅供您本地檢視）

這些資料不會離開您的裝置，也不會與開發者或任何第三方同步。解除安裝擴充功能即會一併清除。

四、權限用途

本擴充功能僅請求達成核心功能所需的最小權限。詳見 Chrome Web Store 頁面上的權限說明。

五、與 Google 的關係

本擴充功能為獨立之第三方開源專案，與 Google、Alphabet 或 NotebookLM 產品團隊並無任何關聯、背書或合作關係。

六、兒童隱私

本擴充功能並非為 13 歲以下兒童設計，亦不會特意收集兒童資料。

七、政策變更

若本政策有重大變更，我們會在此頁面與 GitHub 儲存庫公告修改日期。

八、聯絡方式

如有任何問題，請透過 GitHub Issues 與我們聯繫：
https://github.com/huan5678/NotebookLM-OmniBridge/issues

Privacy Policy — NotebookLM Omni-Bridge

Last updated: 2026-04-21

NotebookLM Omni-Bridge (the "Extension") is an open-source Chrome extension that sends the user's currently viewed web page content to Google NotebookLM and provides a chat interface in Chrome's side panel.

1. We do not collect any personal data

  • The Extension has no developer-operated server and transmits no data to any third party.
  • We do not collect, store, or transmit: names, emails, IP addresses, browsing history, device identifiers, or any personally identifiable information.
  • We do not use analytics, telemetry, advertising identifiers, or any tracking tools.

2. Data flow

All network requests are sent directly to Google NotebookLM (https://notebooklm.google.com), authenticated using the Google session cookies already present in your Chrome browser — equivalent to visiting NotebookLM in a tab yourself.

The following is sent from your browser to Google:

  • When you click "ingest": the current page's title, URL, and text content (or your selected text)
  • Messages you type in the chat
  • Files you explicitly choose to upload

This data is handled by Google under its own Privacy Policy. The Extension's developer has no access to it.

3. Local storage

The Extension uses chrome.storage to persist the following only on your local browser:

  • User preferences (theme, default Notebook, UI language, etc.)
  • Chat history (for your own local reference)

This data never leaves your device and is removed when you uninstall the Extension.

4. Permission usage

The Extension requests only the minimum permissions required for its core functionality. See the Chrome Web Store listing for per-permission justifications.

5. Relationship with Google

This Extension is an independent, third-party open-source project. It is not affiliated with, endorsed by, or sponsored by Google, Alphabet, or the NotebookLM product team.

6. Children's privacy

The Extension is not directed at children under 13 and does not knowingly collect data from children.

7. Changes to this policy

Material changes will be announced on this page and in the GitHub repository with an updated date.

8. Contact

For questions, please open an issue at
https://github.com/huan5678/NotebookLM-OmniBridge/issues.
